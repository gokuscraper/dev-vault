Cache-Control Directives

DirectiveEffectWhere
max-age=NCache for N seconds from response timeResponse
s-maxage=NShared cache (CDN) max age, overrides max-ageResponse
no-cacheMust revalidate before use (NOT "don't cache"!)Both
no-storeNever store in any cacheBoth
publicCan be cached by shared caches (CDNs)Response
privateOnly browser cache, not CDNsResponse
must-revalidateMust revalidate once stale (no serving stale)Response
stale-while-revalidate=NServe stale for N seconds while revalidating in backgroundResponse
immutableNever revalidate (content will never change at this URL)Response
