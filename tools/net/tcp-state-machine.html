Key Concepts

ConceptExplanation
3-Way HandshakeSYN → SYN-ACK → ACK. Establishes connection and synchronizes sequence numbers
4-Way TerminationFIN → ACK → FIN → ACK. Each side independently closes its half of the connection
TIME_WAIT (2×MSL)Prevents old duplicate segments from being accepted. MSL = Maximum Segment Lifetime (typically 60s)
Half-CloseOne side sends FIN but can still receive data. Used in HTTP pipelining
Simultaneous CloseBoth sides send FIN at the same time → CLOSING state → TIME_WAIT
RST (Reset)Immediate connection abort. No TIME_WAIT. Used for port unreachable, firewall drops
