Regex Quick Reference

Common regular expression syntax and patterns

Anchors

^^helloStart of string
$world$End of string
\b\bword\bWord boundary
\B\BwordNot a word boundary

Character Classes

.h.tAny character except newline
\d\d{3}Digit [0-9]
\w\w+Word char [a-zA-Z0-9_]
\s\sWhitespace [ \t\n\r\f]
\D\DNon-digit
\W\WNon-word char
\S\SNon-whitespace
[abc][aeiou]Any character in the set
[^abc][^0-9]Any character NOT in the set
[a-z][a-f]Range of characters

Quantifiers

*ab*c0 or more (greedy)
+ab+c1 or more (greedy)
?ab?c0 or 1 (greedy)
*?ab*?c0 or more (lazy)
+?ab+?c1 or more (lazy)
{n}\d{3}Exactly n times
{n,}\d{3,}n or more
{n,m}\d{2,4}Between n and m

Groups & Alternation

(abc)(ha)+Capturing group
(?:abc)(?:ha)+Non-capturing group
\1(.)\1Backreference to group 1
|cat|dogAlternation (OR)

Lookahead & Lookbehind

(?=...)(?=\d)Positive lookahead
(?!...)(?!\d)Negative lookahead
(?<=...)(?<=@)Positive lookbehind
(?(?Negative lookbehind

Flags

g/hello/gGlobal (find all matches)
i/hello/iCase-insensitive
m/hello/mMultiline (^ and $ match lines)
s/hello/sDotall (. matches newline)
u/hello/uUnicode mode

Common Patterns

^[a-zA-Z0-9]+$AlphanumericOnly letters and numbers
^[a-zA-Z]+$AlphabeticOnly letters
^\d+$NumericOnly digits
^\w+@\w+\.\w+$Basic emailSimple email pattern
^https?://.+HTTP URLStarts with http:// or https://
^(\+?\d{1,3}[-.\s]?)?\(?\d{3}\)?[-.\s]?\d{3}[-.\s]?\d{4}$US PhoneBasic US phone number
(?=.*[a-z])(?=.*[A-Z])(?=.*\d).{8,}Password strengthAt least 8, one upper, one lower, one digit
^#?([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$Hex color#fff or #ffffff
^(19|20)\d{2}[-/](0[1-9]|1[0-2])[-/](0[1-9]|[12]\d|3[01])$Date (YYYY-MM-DD)Basic date validation
^\d{5}(-\d{4})?$US ZIP code5 or 9-digit ZIP
^((25[0-5]|2[0-4]\d|1\d{2}|[1-9]?\d)\.){3}(25[0-5]|2[0-4]\d|1\d{2}|[1-9]?\d)$IPv4IP address validation
<[^>]+>HTML tagMatches any HTML tag
\s+Whitespace collapseOne or more whitespace chars
