4 CSP Integration

To enforce Trusted Types, add this Content-Security-Policy header. The browser will throw a TypeError for any untyped assignment to dangerous sinks.

Content-Security-Policy: require-trusted-types-for 'script'; trusted-types my-sanitizer
Content-Security-Policy: require-trusted-types-for 'script'; trusted-types my-sanitizer
<meta http-equiv="Content-Security-Policy" content="require-trusted-types-for 'script'; trusted-types my-sanitizer">
add_header Content-Security-Policy "require-trusted-types-for 'script'; trusted-types my-sanitizer" always;

Tip: Start with Content-Security-Policy-Report-Only to test without breaking existing code. Add report-uri /csp-report to collect violations server-side.
