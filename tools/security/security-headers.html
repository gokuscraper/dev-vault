Strict-Transport-Security (HSTS)

Forces browsers to use HTTPS. Protects against protocol downgrade attacks and cookie hijacking.

31536000 (1 year)

X-Content-Type-Options

Prevents MIME type sniffing. Blocks requests if the declared type doesn't match the expected type.

X-Frame-Options

Controls whether the page can be embedded in iframes. Prevents clickjacking attacks.

X-XSS-Protection DEPRECATED

Legacy XSS filter. Modern browsers have removed it, but some older browsers still support it. Set to 0 to disable (recommended) or 1; mode=block for legacy compatibility.

Referrer-Policy

Controls how much referrer information is sent with requests. Protects user privacy.

Permissions-Policy

Controls which browser features and APIs can be used. Restricts access to sensitive device capabilities.

Cross-Origin-Opener-Policy

Isolates the browsing context. Required for SharedArrayBuffer and high-resolution timers.

Cross-Origin-Embedder-Policy

Prevents loading cross-origin resources without explicit permission. Required for cross-origin isolation.

Cross-Origin-Resource-Policy

Restricts which origins can load this resource. Prevents speculative side-channel attacks like Spectre.
