Layout Shift Prevention

See how font-size-adjust prevents Cumulative Layout Shift (CLS) when web fonts load. Click "Simulate Load" to swap from fallback to web font.

Without font-size-adjust

CLS detected
When a web font loads without font-size-adjust, the text reflows because the x-height differs between the fallback and loaded font. This causes visible layout shifts that hurt user experience and Core Web Vitals scores.

With font-size-adjust
