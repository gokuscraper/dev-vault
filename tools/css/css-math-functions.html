calc() Nesting & Type Mixing

calc() enables mixing units: calc(100% - 2rem), nesting functions, and combining different unit types. The result type must be valid for the property.

/* Basic mixing */ width: calc(100% - 40px); /* Nested calc (outer calc optional) */ width: calc(100% - calc(2rem + 10px)); /* Same as: */ width: calc(100% - 2rem - 10px); /* Type mixing rules: length + length = length OK length * number = length OK length / number = length OK length + number = ERROR length * length = ERROR */ /* Complex nesting */ font-size: calc( 1rem + (100vw - 320px) / (1200 - 320) * (24 - 16) ); /* With CSS variables */ --gap: 1rem; --cols: 3; width: calc( (100% - (var(--cols) - 1) * var(--gap)) / var(--cols) );

min() / max() / clamp()

min() picks the smallest value, max() the largest, and clamp(min, preferred, max) constrains a value between bounds. These enable responsive design without media queries.

/* min() — use the smaller value */ width: min(600px, 90vw); /* At 400px viewport: min(600, 360) = 360px */ /* At 800px viewport: min(600, 720) = 600px */ /* max() — use the larger value */ font-size: max(16px, 2vw); /* Ensures text never below 16px */ /* clamp(min, preferred, max) */ font-size: clamp(1rem, 2.5vw, 2rem); /* = max(1rem, min(2.5vw, 2rem)) */ /* Responsive padding */ padding: clamp(1rem, 5vw, 3rem); /* Responsive container */ width: min(1200px, 100% - 2rem);

round() / mod() / rem()

CSS Values 4

Stepped value functions for snapping to grids, creating discrete layouts, and rhythmic spacing.

/* round(strategy, value, interval) */ /* Strategies: nearest, up, down, to-zero */ width: round(100px, 30px); /* = round(nearest, 100px, 30px) = 90px */ width: round(up, 100px, 30px); /* = 120px (next multiple of 30) */ width: round(down, 100px, 30px); /* = 90px (prev multiple of 30) */ /* mod() — CSS modulo */ /* mod(value, divisor) */ width: mod(350px, 100px); /* = 50px (remainder) */ /* rem() — CSS remainder */ /* rem(value, divisor) */ /* Same sign as dividend (not divisor) */ width: rem(-18px, 5px); /* = -3px (sign of -18) */ /* Use case: snap to 8px grid */ padding: round(nearest, 1.5rem, 8px);

sin() / cos() / tan() — Circular Patterns

CSS trigonometric functions accept angles and return unitless numbers. Perfect for circular layouts, wave effects, and animation paths.

/* CSS trig takes angles, returns numbers */ /* sin(90deg) = 1, cos(0deg) = 1 */ /* Circular layout */ .item:nth-child(1) { --angle: calc(360deg / 8 * 0); left: calc(50% + cos(var(--angle)) * 40%); top: calc(50% + sin(var(--angle)) * 40%); } /* Wave pattern */ .bar:nth-child(n) { --i: attr(data-i type(<number>)); height: calc( 50% + sin(var(--i) * 30deg + var(--phase)) * 40% ); } /* Also available: atan(), asin(), acos(), atan2() */ --angle: atan2(y, x); /* angle from coordinates */

pow() / sqrt() / log() / exp()

Exponential and logarithmic functions for growth curves, easing, and non-linear scaling.

/* pow(base, exponent) */ --scale: pow(2, 3); /* 8 */ --scale: pow(1.5, var(--level)); /* exponential growth */ /* sqrt(value) */ --diag: calc(sqrt(2) * 100px); /* diagonal of 100px square */ /* log(value) — natural log */ /* log(value, base) — log with base */ --log2: log(8, 2); /* 3 */ --ln: log(2.718); /* ~1 */ /* exp(value) — e^value */ --e2: exp(2); /* ~7.389 */ /* Constants */ /* e = 2.718..., pi = 3.14159... */ --circumference: calc(2 * pi * var(--radius)); /* infinity, -infinity, NaN also available */

abs() / sign()

abs() returns the absolute value. sign() returns -1, 0, or 1. Useful for direction-aware styling and distance calculations.

/* abs(value) — always positive */ --distance: abs(var(--x) - 50%); /* Distance from center, regardless of side */ /* sign(value) — direction indicator */ --dir: sign(var(--scroll-offset)); /* -1 if scrolled up, 0 at top, 1 if scrolled down */ /* Use case: symmetrical effects */ .item { --offset: calc(var(--i) - var(--center)); --dist: abs(var(--offset)); opacity: calc(1 - var(--dist) * 0.15); scale: calc(1 - var(--dist) * 0.1); } /* Use case: directional transforms */ .arrow { --dx: calc(var(--target-x) - var(--x)); rotate: calc(sign(var(--dx)) * 45deg); }
