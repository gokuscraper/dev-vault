Multiplication Steps

CSS Transform Functions → Matrix

  • translate(tx, ty) → matrix(1, 0, 0, 1, tx, ty)
  • scale(sx, sy) → matrix(sx, 0, 0, sy, 0, 0)
  • rotate(θ) → matrix(cos θ, sin θ, -sin θ, cos θ, 0, 0)
  • skewX(θ) → matrix(1, 0, tan θ, 1, 0, 0)
  • skewY(θ) → matrix(1, tan θ, 0, 1, 0, 0)

2D Matrix Format

matrix(a, b, c, d, e, f) represents:

| a  c  e |
| b  d  f |
| 0  0  1 |

Where (a,b) = x-axis direction, (c,d) = y-axis direction, (e,f) = translation.

Matrix Multiplication

When you chain transforms like transform: rotate(45deg) scale(2), the browser multiplies the matrices right to left. Scale is applied first, then rotation.

This tool shows the multiplication step by step so you can see how each transform compounds.

Transform Order Matters

rotate(45deg) translate(100px, 0) ≠ translate(100px, 0) rotate(45deg)

The first rotates the coordinate system, then translates along the rotated axis. The second translates first, then rotates around the original origin.
