1. Basic Style Query

Toggle a CSS custom property on the parent — the child responds via @container style(--highlight: true)

Interactive Demo

Child Element

This element reacts to the parent's --highlight custom property — no classes toggled on the child, no JavaScript styling.

/* Parent sets the custom property */ .parent { --highlight: false; } .parent.active { --highlight: true; } /* Child responds — no container-type needed for style queries */ @container style(--highlight: true) { .child { background: linear-gradient(...); transform: translateY(-2px); } }

3. Theme Switching

Switch between dark, light, and colorful themes with @container style(--theme: ...)

Theme Toggle

Adaptive Card Component

This card's appearance is entirely controlled by the parent container's --theme property. No class changes on the card itself.

.container { --theme: dark; } @container style(--theme: dark) { .card { background: #1e293b; } } @container style(--theme: light) { .card { background: #f8fafc; } } @container style(--theme: colorful) { .card { background: linear-gradient(135deg, #312e81, #581c87); } }

Card Variant
🎨

Creative Project

.container { --variant: default; } @container style(--variant: horizontal) { .card { flex-direction: row; } .card-media { width: 160px; height: auto; } } @container style(--variant: compact) { .card { flex-direction: row; align-items: center; } .card-media { width: 48px; height: 48px; } .card p { display: none; } }

5. Multiple Conditions (AND Logic)

Combine style queries with and — both conditions must match: @container style(--size: large) and style(--priority: high)

Alert Component

/* Both conditions must match */ @container style(--size: large) and style(--priority: high) { .item { border-color: var(--red); padding: 1.5rem; font-size: 1.1rem; } }

Note: Style queries currently only support querying CSS custom properties (e.g., --theme). Querying standard CSS properties like background-color or display is not yet implemented in any browser. Firefox remains the last major holdout — Interop 2025/2026 may bring it. No container-type declaration is needed for style queries (unlike size queries).
