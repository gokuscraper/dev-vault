🎨 CSS Custom Properties as Paint Inputs

      
Paint worklets can read CSS custom properties via inputProperties. This allows CSS to control paint parameters without JavaScript.

📝 Worklet Code

        
class DotPainter {
  static get inputProperties() {
    return [
      '--dot-color',
      '--dot-size',
      '--dot-spacing',
      '--dot-opacity'
    ];
  }

  paint(ctx, geom, properties) {
    const color = properties.get('--dot-color')
      .toString().trim() || '#3b82f6';
    const size = parseInt(properties.get('--dot-size'))
      || 8;
    const spacing = parseInt(
      properties.get('--dot-spacing')) || 20;
    const opacity = parseFloat(
      properties.get('--dot-opacity')) || 1;

    ctx.globalAlpha = opacity;
    ctx.fillStyle = color;

    for (let x = spacing; x < geom.width;
         x += spacing) {
      for (let y = spacing; y < geom.height;
           y += spacing) {
        ctx.beginPath();
        ctx.arc(x, y, size / 2, 0, Math.PI * 2);
        ctx.fill();
      }
    }
  }
}

registerPaint('dots', DotPainter);


        
CSS Usage

        
.element {
  --dot-color: #3b82f6;
  --dot-size: 8;
  --dot-spacing: 20;
  --dot-opacity: 1;
  background-image: paint(dots);
}

      

    

  


  
  

    

      
🎬 Animation with CSS Custom Properties

      
By registering custom properties with @property and using CSS transitions/animations, you can animate paint worklet parameters smoothly.

@property Registration

        
/* Register for smooth CSS animation */
@property --wave-offset {
  syntax: '<number>';
  inherits: false;
  initial-value: 0;
}

@keyframes wave {
  to { --wave-offset: 1; }
}

.animated {
  background-image: paint(wave);
  animation: wave 2s linear infinite;
}

/* The paint worklet reads --wave-offset
   and uses it to shift the wave phase */

      

    

  


  
  

    

      
📚 CSS Paint API Reference

      
The CSS Paint API (part of CSS Houdini) lets you programmatically generate images for CSS properties that expect images, such as background-image, border-image, and mask-image.

    


    

      
registerPaint()

      
// paint-worklet.js
class MyPainter {
  // Declare which CSS properties this worklet reads
  static get inputProperties() {
    return ['--my-color', '--my-size'];
  }

  // Declare which arguments paint() accepts from CSS
  static get inputArguments() {
    return ['<color>', '<length>'];
  }

  // Whether the paint depends on context (alpha, etc.)
  static get contextOptions() {
    return { alpha: true };
  }

  // The paint function — called every repaint
  paint(ctx, geom, properties, args) {
    // ctx: PaintRenderingContext2D (subset of Canvas2D)
    // geom: { width, height } of the element
    // properties: StylePropertyMapReadOnly
    // args: List of CSSStyleValue from CSS paint() args

    const color = properties.get('--my-color').toString();
    ctx.fillStyle = color;
    ctx.fillRect(0, 0, geom.width, geom.height);
  }
}

registerPaint('my-paint', MyPainter);

    


    

      
Registration & Usage

      
// main.js — Register the worklet
if ('paintWorklet' in CSS) {
  CSS.paintWorklet.addModule('paint-worklet.js');
}

// Alternative: inline worklet via Blob URL
const workletCode = `
  class MyPaint {
    paint(ctx, geom) {
      ctx.fillStyle = 'blue';
      ctx.fillRect(0, 0, geom.width, geom.height);
    }
  }
  registerPaint('my-paint', MyPaint);
`;
const blob = new Blob([workletCode],
  { type: 'application/javascript' });
CSS.paintWorklet.addModule(URL.createObjectURL(blob));

      
/* CSS */
.element {
  --my-color: #3b82f6;
  --my-size: 10;
  background-image: paint(my-paint);

  /* Also works with: */
  /* border-image: paint(my-paint); */
  /* mask-image: paint(my-paint); */
}

Paint worklets are only supported in Chromium-based browsers. Always provide a CSS fallback for other browsers using @supports(background: paint(id)).

    


    

      
PaintRenderingContext2D

      
The paint context is a subset of CanvasRenderingContext2D. Available methods:

      

        
fillRect()

        
strokeRect()

        
clearRect()

        
beginPath() / closePath()

        
moveTo() / lineTo()

        
arc() / arcTo()

        
bezierCurveTo()

        
quadraticCurveTo()

        
fill() / stroke()

        
createLinearGradient()

        
createRadialGradient()

        
save() / restore()

        
translate() / rotate()

        
scale() / transform()

        
❌ fillText() (not available)

        
❌ drawImage() (not available)
