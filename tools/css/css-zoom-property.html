Basic Demo

The CSS zoom property scales an element and affects layout — surrounding content reflows to accommodate the scaled size. This is the key difference from transform: scale().

Nested Zoom

When zoom is applied to nested elements, the values multiply. A child at zoom: 1.5 inside a parent at zoom: 2 renders at an effective 3× scale.

Browser Support

zoom is now part of Interop 2025 and standardized in CSS. It has been supported in Chromium and WebKit for years, and Firefox added support in version 126 (June 2024).

Important differences from transform: scale():
1. zoom affects layout — the element's box size changes for its siblings
2. zoom is not animatable via CSS transitions/animations (use transform for animations)
3. zoom: normal resets to the initial zoom (equivalent to zoom: 1)
4. zoom values compound when nested (multiplicative)
