Quick Reference

Syntax patterns for CSS Relative Color Syntax.

PatternDescription
oklch(from <color> l c h)Identity (same color in oklch)
oklch(from <color> calc(l * 1.2) c h)20% lighter
oklch(from <color> calc(l * 0.8) c h)20% darker
oklch(from <color> l calc(c * 1.5) h)50% more chroma (vibrant)
oklch(from <color> l calc(c * 0.5) h)50% less chroma (muted)
oklch(from <color> l c calc(h + 180))Complementary (hue +180)
oklch(from <color> l c calc(h + 30))Analogous neighbor (+30)
oklch(from <color> l c h / 0.5)50% opacity
hsl(from <color> h s calc(l + 20%))Add 20% lightness in HSL
rgb(from <color> calc(r + 50) g b)Add 50 to red channel
lab(from <color> calc(l + 10) a b)Increase L* by 10 in Lab
lch(from <color> l c calc(h - 90))Rotate hue -90 in LCH
oklch(from <color> l 0 h)Desaturate to gray (chroma = 0)
oklch(from <color> 0.5 c h)Set lightness to 50%

Color SpaceChannelsRanges
