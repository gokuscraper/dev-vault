shape() vs polygon() vs path()

shape() improves on existing CSS path methods with human-readable syntax and CSS unit support.

Feature polygon() path() shape()
Syntax polygon(x y, ...) path("M 0 0 L...") shape(from x y, line to ...)
Readability Good for simple shapes Cryptic SVG letters Best — named commands
CSS Units Yes (%, px, etc.) No (unitless px only) Yes (%, px, em, etc.)
Curves No Yes Yes (arc, curve, smooth)
Responsive With % No (fixed px) Yes (any CSS unit)
Arcs No Yes (A command) Yes (arc to)
Interpolation Same # points Same # segments Same # commands
Browser Support All modern All modern Chrome/Edge 139+
