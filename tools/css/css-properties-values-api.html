Animated Custom Properties

      
Unlike untyped custom properties, @property-registered properties can be smoothly animated and transitioned.

Continuous Animation

      

      

        
        
        
      

    


    

      
Generated Animation CSS

      
/* Register properties for animation */
@property --grad-start {
  syntax: "<color>";
  inherits: false;
  initial-value: #7c6aef;
}
@property --grad-end {
  syntax: "<color>";
  inherits: false;
  initial-value: #ff6b6b;
}

.element {
  --grad-start: #7c6aef;
  --grad-end: #ff6b6b;
  background: linear-gradient(135deg, var(--grad-start), var(--grad-end));
  transition: --grad-start 1s ease, --grad-end 1s ease;
}
.element:hover {
  --grad-start: #ff6b6b;
  --grad-end: #4ecdc4;
}
