@container scroll-state()

Interactive playground for CSS Scroll State Container Queries. Style elements based on sticky, snap, and scrollable states — no JavaScript needed.

Uses scroll-state(stuck: top) to restyle a sticky header when it adheres to the scroll container edge. Scroll down to see the effect.

Sticky Sidebar

  
Uses scroll-state(stuck: top) on a sidebar element. The sidebar stays fixed while main content scrolls, with a visual indicator when stuck.
