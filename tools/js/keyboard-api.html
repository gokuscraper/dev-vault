Keyboard Lock

Lock specific keys so the browser captures them instead of the OS. Requires fullscreen in most browsers.

Lock Control

Unlocked

Select keys to lock, then click Lock. Most browsers require fullscreen mode.

// Lock specific keys
await navigator.keyboard.lock(['Escape', 'KeyW', 'KeyA', 'KeyS', 'KeyD']);

// Lock all keys
await navigator.keyboard.lock();

// Unlock
navigator.keyboard.unlock();
