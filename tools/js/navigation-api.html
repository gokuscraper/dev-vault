🎯 Navigation Types

Different navigation types: push adds a new entry, replace replaces the current entry, and hash changes only the fragment.

Push Navigation — Adds a new entry to the history stack.

navigation.navigate('/new-page', {
  history: 'push',
  state: { timestamp: Date.now() }
});

Replace Navigation — Replaces the current entry without adding to the stack.

navigation.navigate('/replaced', {
  history: 'replace',
  state: { replaced: true }
});

Hash Navigation — Changes only the URL fragment. Does not create a full navigation in some cases.

navigation.navigate('#section-' + Math.random().toString(36).slice(2, 8));

Reload Navigation — Reloads the current entry. navigationType will be "reload".

navigation.reload({
  state: { reloadedAt: Date.now() }
});

Traverse Navigation — Move to a specific history entry by key.

// First navigate to create entries, then traverse back
const entries = navigation.entries();
if (entries.length > 1) {
  navigation.traverseTo(entries[0].key);
}
