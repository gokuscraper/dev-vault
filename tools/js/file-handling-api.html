The File Handling API requires a PWA installed on the user's system. The browser registers the file type associations with the OS during PWA installation. The file_handlers manifest member defines which types your app handles, and the launchQueue API delivers the files to your code. Works alongside the Launch Handler API for controlling window behavior.
