execCommand Reference

document.execCommand() is deprecated. Use Input Events API (beforeinput/input with inputType) for modern implementations.

Modern Patterns

// Modern approach: beforeinput event editor.addEventListener('beforeinput', (e) => { console.log(e.inputType); // "insertText", "formatBold", etc. console.log(e.data); // inserted text console.log(e.dataTransfer); // for paste events // e.preventDefault() to block the action }); // Selection API const sel = window.getSelection(); const range = sel.getRangeAt(0); const wrapper = document.createElement('mark'); range.surroundContents(wrapper); // Clipboard API (modern) editor.addEventListener('paste', async (e) => { e.preventDefault(); const text = await navigator.clipboard .readText(); document.execCommand('insertText', false, text); // or use insertText inputType }); // Sanitizer API (experimental) const sanitizer = new Sanitizer({ allowElements: ['p', 'b', 'i', 'em', 'strong', 'a'], allowAttributes: { 'href': ['a'] } }); editor.setHTML(dirtyHTML, { sanitizer });
