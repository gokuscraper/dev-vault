💡 Use Case Demos

News Feed Sync

Periodically fetch latest news articles in the background so they're available when the user opens the app.

minInterval: 12 hours (43200000 ms)
// Register const reg = await navigator.serviceWorker.ready; await reg.periodicSync.register('news-sync', { minInterval: 12 * 60 * 60 * 1000 }); // In service worker self.addEventListener('periodicsync', (e) => { if (e.tag === 'news-sync') { e.waitUntil(fetchAndCacheNews()); } });

Content Pre-caching

Keep cached content fresh by periodically updating resources the user frequently accesses.

minInterval: 1 day (86400000 ms)
// Register with daily interval await reg.periodicSync.register('cache-refresh', { minInterval: 24 * 60 * 60 * 1000 }); // SW handler async function refreshCache() { const cache = await caches.open('v1'); const urls = ['/api/data', '/api/config']; await Promise.all( urls.map(u => cache.add(u)) ); }

Email Notifications Badge

Check for new emails and update the app badge count, even when the app is not open.

minInterval: 1 hour (3600000 ms)
await reg.periodicSync.register('email-check', { minInterval: 60 * 60 * 1000 }); // SW: update badge async function checkEmails() { const count = await fetchUnreadCount(); await navigator.setAppBadge(count); }

Analytics Batch Upload

Queue analytics events locally and batch-upload them periodically to reduce network requests.

minInterval: 6 hours (21600000 ms)
await reg.periodicSync.register('analytics-upload', { minInterval: 6 * 60 * 60 * 1000 }); // SW: flush queued events async function uploadAnalytics() { const db = await openDB('analytics'); const events = await db.getAll('queue'); await fetch('/api/analytics', { method: 'POST', body: JSON.stringify(events) }); await db.clear('queue'); }

Key constraints:
  • The browser may adjust the actual interval based on site engagement score
  • minInterval is a minimum - actual intervals may be much longer
  • Only works for installed PWAs (Add to Home Screen / installed web apps)
  • Requires sufficient user engagement with the site
  • The browser can deny or revoke permission at any time
