💡 Adaptive Loading Patterns

Connection-Aware Fetch

Adjust API payloads based on connection quality.

const conn = navigator.connection; const fields = conn.effectiveType === '4g' ? 'full' : 'minimal'; fetch(`/api/data?fields=${fields}`);

Lazy Loading Strategy

Adjust intersection observer margins based on bandwidth.

const margin = conn.effectiveType === '4g' ? '200px' // prefetch early : '50px'; // load just-in-time const obs = new IntersectionObserver( cb, { rootMargin: margin } );

Video Quality Selection

Choose appropriate stream quality for current conditions.

function getVideoQuality() { const dl = navigator.connection.downlink; if (dl >= 10) return '1080p'; if (dl >= 5) return '720p'; if (dl >= 1) return '480p'; return '240p'; }

Save Data Respect

Honor the user's data-saving preference.

if (navigator.connection.saveData) { // Skip non-essential resources document.querySelectorAll('[data-src]') .forEach(img => { if (!img.dataset.essential) img.remove(); }); }

Prefetch Control

Only prefetch resources on fast connections.

if (conn.effectiveType === '4g' && !conn.saveData) { const link = document.createElement('link'); link.rel = 'prefetch'; link.href = '/next-page.html'; document.head.appendChild(link); }

Font Loading Strategy

Use system fonts on slow connections to avoid FOUT.

const useWebFonts = conn.effectiveType === '4g' || conn.effectiveType === '3g'; if (useWebFonts) { loadGoogleFont('Inter'); } // else: system font stack
