💤 Tab Suspension Behavior

What is tab suspension? Modern browsers may freeze or discard background tabs to save memory and CPU.
  • Chrome: Freezes tabs after ~5 minutes in background. Discards tabs under memory pressure (chrome://discards).
  • Edge: "Sleeping tabs" feature freezes inactive tabs after configurable timeout (default 2 hours).
  • Firefox: Tab unloading under memory pressure. No freeze API support.
  • Safari: Aggressive tab suspension on iOS. Tabs can be suspended within seconds on mobile.

Detecting suspension: Compare Date.now() timestamps across setInterval calls. A gap much larger than the interval indicates the tab was frozen.

✅ Best Practices for Each State

Active State

  • Full interactivity enabled
  • All UI updates running
  • Animations and transitions active
  • Input handlers fully responsive
// Nothing special needed — this is the default // Just ensure smooth 60fps rendering

Passive State

  • Page visible but not focused
  • Reduce non-essential CPU work
  • Keep visual updates running
  • Pause non-visible animations
window.addEventListener('blur', () => { // Reduce update frequency clearInterval(heavyTimer); heavyTimer = setInterval(update, 5000); });

Hidden State

  • Stop all unnecessary work
  • Save state to localStorage/IDB
  • Send analytics via sendBeacon
  • Close non-essential connections
document.addEventListener( 'visibilitychange', () => { if (document.hidden) { saveState(); navigator.sendBeacon('/analytics', JSON.stringify(data)); } });

Frozen State

  • No JS execution possible
  • Prepare in 'freeze' event
  • Save all transient state
  • Close IndexedDB connections
document.addEventListener('freeze', () => { // Last chance to save state saveToLocalStorage(appState); // Close IDB connections db.close(); });

Terminated State

  • Page being unloaded
  • Cannot run async operations
  • Use sendBeacon for final data
  • Avoid beforeunload when possible
window.addEventListener('pagehide', (e) => { if (!e.persisted) { // Page will not be in bfcache navigator.sendBeacon( '/log', 'session-end'); } });

Discarded State

  • Tab unloaded by browser to free memory
  • All JS state is lost
  • Restore from localStorage on reload
  • Check document.wasDiscarded
// On page load, check if restored if (document.wasDiscarded) { const saved = localStorage .getItem('appState'); if (saved) restoreState( JSON.parse(saved)); }
