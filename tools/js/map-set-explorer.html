WeakMap & WeakSet

Weak collections hold objects without preventing garbage collection. Keys must be objects (or non-registered symbols). They are not iterable and have no size property — useful for metadata without memory leaks.

WeakRef & FinalizationRegistry

WeakRef holds a weak reference to an object — deref() returns the object or undefined if GC'd. FinalizationRegistry runs a callback after GC. Note: GC timing is non-deterministic.
