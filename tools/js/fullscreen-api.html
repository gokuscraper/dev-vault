🔄 Screen Orientation Lock Integration

Screen orientation can be locked when in fullscreen mode (supported on mobile devices and some desktop browsers). The orientation lock API requires fullscreen mode to be active first.

📝 Fullscreen Compatibility Notes

Element Fullscreen Notes
<div> Supported Most common target. Background fills screen.
<canvas> Supported Ideal for games. Resize canvas on fullscreenchange.
<video> Supported Native controls auto-shown. Some browsers have built-in fullscreen.
<iframe> Conditional Requires allowfullscreen attribute. Cross-origin limits apply.
<dialog> Supported Can go fullscreen when open. ::backdrop stacks with dialog backdrop.
<html> Supported Entire document. Same as F11 in most browsers.
