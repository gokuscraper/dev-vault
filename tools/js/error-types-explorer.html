Error.cause Chaining

ES2022 added the cause option to Error constructors: new Error("msg", { cause: originalError }). This creates a chain of errors for better debugging.

AggregateError

AggregateError wraps multiple errors into one. Used by Promise.any() when all promises reject. The .errors property holds the array of individual errors.

Stack Trace Analysis

Every Error has a .stack property (non-standard but universal). It shows the call chain at the point the error was created. Stack traces are crucial for debugging.
