Basic Usage

// Deep clone any structured-cloneable value const original = { name: "test", nested: { deep: { value: 42 } }, date: new Date(), map: new Map([["key", "value"]]), set: new Set([1, 2, 3]), regex: /pattern/gi, buffer: new ArrayBuffer(8) }; const clone = structuredClone(original); // Verify deep clone (not same reference) console.log(clone.nested === original.nested); // false console.log(clone.nested.deep.value === 42); // true console.log(clone.date instanceof Date); // true console.log(clone.map.get("key")); // "value"

Transfer (Move, Not Copy)

// Transfer ownership of ArrayBuffer (zero-copy) const buffer = new ArrayBuffer(1024); console.log(buffer.byteLength); // 1024 const clone = structuredClone(buffer, { transfer: [buffer] }); console.log(clone.byteLength); // 1024 console.log(buffer.byteLength); // 0 — original is neutered! // Useful for Web Workers: // worker.postMessage(data, [data.buffer]);

What Throws

// These all throw DataCloneError: structuredClone(() => {}); // Function structuredClone(Symbol("x")); // Symbol structuredClone(document.body); // DOM Node structuredClone({ get x() { return 1; } }); // Getter → value preserved, getter lost structuredClone(new WeakMap()); // WeakMap structuredClone(new WeakRef({})); // WeakRef structuredClone(new Proxy({}, {})); // Proxy // Workaround: strip non-cloneable properties first function safeClone(obj) { try { return structuredClone(obj); } catch { return JSON.parse(JSON.stringify(obj)); // fallback (lossy) } }

vs JSON.parse(JSON.stringify())

const obj = { date: new Date(), map: new Map([["a", 1]]), set: new Set([1, 2]), regex: /test/g, undef: undefined, nan: NaN, inf: Infinity, buffer: new ArrayBuffer(8) }; // JSON round-trip: LOSES types const jsonClone = JSON.parse(JSON.stringify(obj)); // date → string, map → {}, set → {}, regex → {} // undefined → missing, NaN → null, Infinity → null // buffer → {} // structuredClone: PRESERVES types const scClone = structuredClone(obj); // date → Date, map → Map, set → Set, regex → RegExp // undefined → undefined, NaN → NaN, Infinity → Infinity // buffer → ArrayBuffer
