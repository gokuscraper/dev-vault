Basic PiP Window

Beginner

Open a PiP window with custom HTML content. The simplest use case: move any DOM element into an always-on-top floating window.

No PiP window open
View Code 
// Request a Document PiP window
const pipWindow = await documentPictureInPicture
  .requestWindow({
    width: 400,
    height: 300,
  });

// Move content into the PiP window
const content = document.getElementById('myContent');
pipWindow.document.body.append(content);

// Copy styles to PiP window
const styles = document.querySelectorAll('style, link[rel="stylesheet"]');
styles.forEach(s => pipWindow.document.head.append(s.cloneNode(true)));

// Listen for PiP window close
pipWindow.addEventListener('pagehide', () => {
  // Move content back to main page
  document.getElementById('container').append(content);
});

const pipWindow = await documentPictureInPicture
  .requestWindow({ width: 320, height: 180 });

// Inject styles into PiP
const style = document.createElement('style');
style.textContent = `
  body { margin:0; font-family:system-ui; background:#1a1d27; color:#e4e7ef; }
  .player { padding:1rem; display:flex; flex-direction:column; gap:0.5rem; }
  .controls { display:flex; justify-content:center; gap:1rem; }
  button { background:none; border:none; color:#e4e7ef; font-size:1.5rem; cursor:pointer; }
`;
pipWindow.document.head.append(style);

// Build player UI in PiP
const player = pipWindow.document.createElement('div');
player.className = 'player';
player.innerHTML = `
  <div class="track">Now Playing: Song Title</div>
  <div class="controls">
    <button onclick="prev()">⏮</button>
    <button onclick="toggle()">▶</button>
    <button onclick="next()">⏭</button>
  </div>
`;
pipWindow.document.body.append(player);

Timer / Stopwatch PiP

An always-on-top timer or stopwatch. Perfect for presentations, cooking, or time-boxed work sessions.

00:00:00
Stopwatch ready
View Code 
const pipWindow = await documentPictureInPicture
  .requestWindow({ width: 300, height: 120 });

const style = document.createElement('style');
style.textContent = `
  body { margin:0; background:#0f1117; color:#ff6b9d;
         font-family:'SF Mono',monospace; display:flex;
         align-items:center; justify-content:center; height:100vh; }
  .time { font-size:3rem; font-weight:700; letter-spacing:2px; }
`;
pipWindow.document.head.append(style);

const el = pipWindow.document.createElement('div');
el.className = 'time';
pipWindow.document.body.append(el);

let seconds = 0;
const interval = setInterval(() => {
  seconds++;
  const h = String(Math.floor(seconds / 3600)).padStart(2, '0');
  const m = String(Math.floor((seconds % 3600) / 60)).padStart(2, '0');
  const s = String(seconds % 60).padStart(2, '0');
  el.textContent = `${h}:${m}:${s}`;
}, 1000);

pipWindow.addEventListener('pagehide', () => clearInterval(interval));

Floating Notes PiP

Productivity

A floating notepad that stays on top. Take quick notes while working in other windows. Content syncs back when closed.

Notes ready
View Code 
const pipWindow = await documentPictureInPicture
  .requestWindow({ width: 350, height: 400 });

const style = document.createElement('style');
style.textContent = `
  body { margin:0; background:#1a1d27; font-family:system-ui; }
  textarea { width:100%; height:100vh; box-sizing:border-box;
             background:#1a1d27; color:#e4e7ef; border:none;
             padding:1rem; font-size:0.9rem; resize:none;
             font-family:inherit; line-height:1.6; }
  textarea:focus { outline:none; }
`;
pipWindow.document.head.append(style);

const textarea = pipWindow.document.createElement('textarea');
textarea.value = document.getElementById('notesContent').value;
textarea.placeholder = 'Type your notes...';
pipWindow.document.body.append(textarea);

// Sync notes back when PiP closes
pipWindow.addEventListener('pagehide', () => {
  document.getElementById('notesContent').value = textarea.value;
});

Window Customization

API Options

Control window dimensions and placement. The API accepts width, height, and preferInitialWindowPlacement options.

Window will be 400 x 300 pixels
View Code 
// All available options for requestWindow()
const pipWindow = await documentPictureInPicture.requestWindow({
  width: 400,        // Initial width in pixels
  height: 300,       // Initial height in pixels

  // Chrome 129+: controls initial placement
  // "enterpictureinpicture" reuses last PiP position
  preferInitialWindowPlacement: "enterpictureinpicture",
});

// The returned window object is a standard Window
console.log(pipWindow.innerWidth);  // actual width
console.log(pipWindow.innerHeight); // actual height

// You can also access the existing PiP window:
const existing = documentPictureInPicture.window;
if (existing) {
  console.log('PiP window is open');
}

Event Handling

API Events

Monitor PiP window lifecycle events: enter on the API object, plus pagehide and resize on the PiP window.

[--:--:--] Waiting for events...
View Code 
// 1. Listen for PiP window creation (on the API object)
documentPictureInPicture.addEventListener('enter', (event) => {
  const pipWindow = event.window;
  console.log('PiP window opened', pipWindow.innerWidth, pipWindow.innerHeight);

  // 2. Listen for PiP window close
  pipWindow.addEventListener('pagehide', () => {
    console.log('PiP window closed');
    // Clean up: move content back, stop intervals, etc.
  });

  // 3. Listen for PiP window resize
  pipWindow.addEventListener('resize', () => {
    console.log('PiP resized:', pipWindow.innerWidth, pipWindow.innerHeight);
  });
});

// Check if a PiP window is currently open
if (documentPictureInPicture.window) {
  console.log('A PiP window exists');
}

Browser Compatibility

Reference

The Document Picture-in-Picture API is relatively new. It extends the existing video PiP API to support arbitrary HTML content.

BrowserSupportSinceNotes
Chrome Supported 116 (Aug 2023) Full support
Edge Supported 116 (Aug 2023) Chromium-based, full support
Opera Supported 102 (Sep 2023) Chromium-based
Firefox Not supported -- No current plans announced
Safari Not supported -- No current plans announced
Chrome Android Not supported -- Desktop only

Key limitations:

  • Desktop Chromium browsers only (Chrome 116+, Edge 116+, Opera 102+)
  • Requires user gesture to open (click, keyboard, etc.)
  • Only one PiP window at a time per page
  • PiP window has same origin as opener
  • No navigation allowed in PiP window
  • preferInitialWindowPlacement requires Chrome 129+
View Feature Detection Code 
// Feature detection
if ('documentPictureInPicture' in window) {
  console.log('Document PiP API is supported!');

  // Check if a PiP window is already open
  if (documentPictureInPicture.window) {
    console.log('A PiP window is currently open');
  }
} else {
  console.log('Document PiP API is NOT supported');
  console.log('Use Chrome 116+ or Edge 116+');
}

// Fallback: check for video PiP at least
if (document.pictureInPictureEnabled) {
  console.log('Video PiP is available (fallback)');
}
