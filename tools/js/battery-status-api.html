💡 Power-Aware Application Patterns

Reduce Polling Frequency

When battery is low, increase intervals for non-critical updates.

const interval = battery.level < 0.2 ? 60000 // 1 min when low : 10000; // 10s normally

Defer Heavy Operations

Postpone background sync, prefetching, or video preloading.

if (battery.charging || battery.level > 0.5) { prefetchNextPage(); }

Adaptive Media Quality

Lower video resolution or disable autoplay on low battery.

const quality = battery.level > 0.3 ? 'high' : 'low'; video.src = `/stream?q=${quality}`;

Dark Mode Activation

Switch to dark theme on OLED screens when battery is low to save power.

if (battery.level < 0.15 && !battery.charging) { document.body.classList .add('power-saver'); }

Notification Strategy

Batch notifications instead of pushing individually when battery is limited.

const strategy = battery.level < 0.2 ? 'batch' : 'immediate'; notifier.setMode(strategy);

Analytics Throttling

Reduce analytics event frequency or defer sending when not charging.

if (!battery.charging && battery.level < 0.3) { analytics.setFlushInterval( 300000 // 5 min batches ); }
