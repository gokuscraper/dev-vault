Read and write multiple MIME types, images, rich text, and custom web formats with the async Clipboard API.

✎ Multi-MIME Writer

Write text/plain + text/html + image/png simultaneously as a single ClipboardItem.

const item = new ClipboardItem({
  'text/plain': new Blob([plainText], {type: 'text/plain'}),
  'text/html': new Blob([htmlText], {type: 'text/html'}),
  'image/png': pngBlob  // optional
});
await navigator.clipboard.write([item]);

🔍 Clipboard Inspector

Read the clipboard and inspect all available MIME types, blob sizes, and content previews.

const items = await navigator.clipboard.read();
for (const item of items) {
  for (const type of item.types) {
    const blob = await item.getType(type);
    console.log(type, blob.size, 'bytes');
  }
}

🎨 Image Copy / Paste

Draw on the canvas or upload an image. Copy it to clipboard, then paste it back.

// Copy canvas as PNG
canvas.toBlob(async blob => {
  const item = new ClipboardItem({'image/png': blob});
  await navigator.clipboard.write([item]);
}, 'image/png');

// Paste image
const [item] = await navigator.clipboard.read();
const blob = await item.getType('image/png');
const img = new Image();
img.src = URL.createObjectURL(blob);

🔧 Custom Web Formats

Write and read custom formats using the "web " prefix. Apps can define their own clipboard types.

// Write custom format (requires "web " prefix)
const type = 'web application/x-my-app';
const item = new ClipboardItem({
  [type]: new Blob([json], {type}),
  'text/plain': new Blob([fallback], {type: 'text/plain'})
});
await navigator.clipboard.write([item]);

// Read back
const blob = await item.getType(type);

✅ ClipboardItem.supports()

Check which MIME types your browser supports for clipboard read/write operations.

// Check if a MIME type is supported
if (typeof ClipboardItem.supports === 'function') {
  const ok = ClipboardItem.supports('image/png');
  console.log('image/png:', ok);
}
// Note: supports() is Chromium 121+

🛡 Copy Event Interception

Select text below and copy (Ctrl+C). The copy event is intercepted to customize what gets placed on the clipboard.

The Clipboard API provides the ability to respond to clipboard commands (cut, copy, and paste), as well as to asynchronously read from and write to the system clipboard. Access to the contents of the clipboard is gated behind the Permissions API. Contact: admin@example.com or support@example.org for questions. This text is selectable. Try selecting a portion and pressing Ctrl+C / Cmd+C to see the interception in action. 
document.addEventListener('copy', e => {
  const sel = window.getSelection().toString();
  e.preventDefault();
  e.clipboardData.setData('text/plain', sel.toUpperCase());
  e.clipboardData.setData('text/html',
    '<b>' + sel.toUpperCase() + '</b>');
});
