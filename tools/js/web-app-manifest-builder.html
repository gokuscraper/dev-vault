📱 Display Modes

      
The display mode determines how much browser UI is shown when the app is launched.

    

    

      

        
📱

        
standalone

        
App-like experience. No browser UI. Has own window.

      

      

        
🖥️

        
fullscreen

        
Fills entire screen. No status bar or navigation.

      

      

        
🪟

        
minimal-ui

        
Minimal browser controls. Back, reload buttons.

      

      

        
🌐

        
browser

        
Full browser tab. URL bar and all controls.

Background Color

        
Shown as a splash screen background while the app loads. Also used if the page hasn't loaded yet.

📝 beforeinstallprompt Handler

      
let deferredPrompt;

window.addEventListener('beforeinstallprompt', (e) => {
  // Prevent the mini-infobar from appearing
  e.preventDefault();
  // Stash the event for later use
  deferredPrompt = e;
  // Show your custom install button
  showInstallButton();
});

async function installApp() {
  if (!deferredPrompt) return;

  // Show the browser install prompt
  deferredPrompt.prompt();

  // Wait for the user to respond
  const { outcome } = await deferredPrompt.userChoice;
  console.log('Install outcome:', outcome);
  // outcome: 'accepted' or 'dismissed'

  deferredPrompt = null;
}

window.addEventListener('appinstalled', () => {
  console.log('App installed successfully!');
  deferredPrompt = null;
});
