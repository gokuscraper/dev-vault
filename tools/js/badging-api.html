Badging API Methods

The Badging API provides two simple methods for controlling the app badge.

navigator.setAppBadge(count)

// Set badge with a count await navigator.setAppBadge(42); // Set badge as a flag (dot indicator, no number) await navigator.setAppBadge(); // The method returns a Promise try { await navigator.setAppBadge(5); console.log('Badge set successfully'); } catch (e) { // May throw if not installed as PWA (some browsers) console.error('Badge not supported:', e); }

navigator.clearBadge()

// Remove the badge entirely await navigator.clearBadge(); // Equivalent to: await navigator.setAppBadge(0);

Favicon Badge Fallback

For browsers that don't support the Badging API (or non-PWA contexts), you can draw a badge directly on the favicon using Canvas. This approach updates the browser tab icon in real-time.

64x64 Preview

128x128 Preview (zoomed)

Check your browser tab to see the favicon badge in action! The favicon is updated dynamically via a data URL.

Current badge: None

Implementation Code

function setFaviconBadge(count, color = '#f44336') { const canvas = document.createElement('canvas'); canvas.width = 64; canvas.height = 64; const ctx = canvas.getContext('2d'); // Draw base icon (could be an image) ctx.fillStyle = '#4285f4'; ctx.fillRect(4, 4, 56, 56); if (count > 0) { // Draw badge circle const text = count > 99 ? '99+' : String(count); const radius = 16; ctx.beginPath(); ctx.arc(48, 16, radius, 0, Math.PI * 2); ctx.fillStyle = color; ctx.fill(); // Draw badge text ctx.fillStyle = '#fff'; ctx.font = `bold ${radius}px sans-serif`; ctx.textAlign = 'center'; ctx.textBaseline = 'middle'; ctx.fillText(text, 48, 16); } // Update favicon const link = document.querySelector("link[rel='icon']") || document.createElement('link'); link.rel = 'icon'; link.href = canvas.toDataURL(); document.head.appendChild(link); }

PWA Manifest Requirements

The Badging API works best with installed PWAs. Here's the minimal manifest configuration needed to make your app installable and support badges.

manifest.json

{ "name": "My App", "short_name": "MyApp", "start_url": "/", "display": "standalone", "background_color": "#0a0a0a", "theme_color": "#ff5722", "icons": [ { "src": "/icon-192.png", "sizes": "192x192", "type": "image/png" }, { "src": "/icon-512.png", "sizes": "512x512", "type": "image/png" } ] }

HTML Setup

<!-- In your HTML <head> --> <link rel="manifest" href="/manifest.json"> <meta name="theme-color" content="#ff5722">

Service Worker (Required for PWA)

// Register service worker for installability if ('serviceWorker' in navigator) { navigator.serviceWorker.register('/sw.js'); } // Set badge from service worker (works in background!) // In sw.js: self.addEventListener('push', (event) => { const data = event.data.json(); // Update badge count from push notification navigator.setAppBadge(data.unreadCount); });
