API Reference
// Check permission const permission = await IdleDetector.requestPermission(); // "granted" | "denied" // Create and start detector const detector = new IdleDetector(); detector.addEventListener('change', () => { const userState = detector.userState; // "active" | "idle" const screenState = detector.screenState; // "locked" | "unlocked" console.log(`User: ${userState}, Screen: ${screenState}`); }); await detector.start({ threshold: 60000, // minimum 60,000ms (60 seconds) signal: controller.signal // AbortSignal for stopping }); // To stop detection controller.abort();
Privacy considerations: The Idle Detection API requires the "idle-detection" permission. Browsers may show a permission prompt. The minimum threshold is 60 seconds to prevent fine-grained user tracking. This API is only available in secure contexts (HTTPS).
