Character Classes

\S Non-whitespace
[a-z] Char range a to z
[^abc] Not a, b, or c
. Any char (except newline)

Quantifiers

* 0 or more
+ 1 or more
? 0 or 1
{n} Exactly n
{n,} n or more
{n,m} Between n and m
*? Lazy (non-greedy)

Anchors

$ End of string/line
\b Word boundary

Groups & Alternation

(...) Capturing group
(?:...) Non-capturing group

(?=...) Positive lookahead
