ARIA Landmark Reference

▶
banner → <header>
Site-wide header. Use once per page (top-level).
navigation → <nav>
Groups of navigation links. Label each with aria-label if multiple.
main → <main>
Primary content of the page. Use once per page.
contentinfo → <footer>
Site-wide footer with copyright, links, contact info.
complementary → <aside>
Supporting content related to the main content (sidebars).
search → <search>
Search functionality. HTML <search> element is new in HTML5.
form → <form>
Form landmark when the form has an accessible name (aria-label/labelledby).
region → <section>
Generic landmark when section has an accessible name. Use sparingly.
